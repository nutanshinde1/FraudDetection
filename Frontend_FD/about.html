About FraudDetectAI

This prototype demonstrates a frontend for a fraud detection platform. Backend (Flask/Django) can be connected to receive CSVs, run ML scoring, and return results. Key ideas:

  • Real-time scoring & alerting
  • Explainable ML features to interpret decisions
  • Exportable reports & admin controls
